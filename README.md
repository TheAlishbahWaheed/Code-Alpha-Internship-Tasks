"# Code-Alpha-Internship-Tasks" 

## About
This repository contains the completed internship tasks, including multiple Python-based projects developed to strengthen programming, problem-solving, and automation skills.

## Projects Included

### 1. Hangman Game
An interactive word-guessing game built using Python.  
Features:
- Random word selection
- Limited attempts system
- User-friendly gameplay
- Win/Loss conditions

### 2. Rule-Based Chatbot
A simple chatbot that responds to user inputs using predefined rules and patterns.  
Features:
- Keyword-based responses
- Easy-to-extend logic
- Lightweight and fast execution

### 3. Stock Portfolio Tracker
A program designed to track and manage stock investments.  
Features:
- Portfolio management
- Basic performance tracking
- Simple financial calculations

### 4. Task Automation Tool
A utility tool to automate repetitive tasks and improve workflow efficiency.  
Features:
- Automates routine operations
- Saves time and effort
- Easy customization

## Technologies Used
- Python 3
- Basic libraries (random, datetime, etc.)
- Command Line Interface (CLI)

## Purpose
The purpose of these projects is to gain hands-on experience in Python programming and understand real-world applications such as automation, AI chat systems, and game development.
